# JD Refrigeração

Landing page estática para a JD Refrigeração.

## Como abrir

Abra o arquivo `index.html` no navegador ou sirva a pasta com qualquer servidor estático.

## Ajustes rápidos

- Troque o número do WhatsApp em `index.html`.
- Atualize depoimentos, se quiser usar relatos reais.
- Se desejar, posso converter este layout para Next.js, React ou WordPress.
